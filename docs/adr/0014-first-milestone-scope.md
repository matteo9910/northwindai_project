# Scope the first milestone around the GraphRAG query ladder

The first implementation milestone will deliver the full GraphRAG query ladder with Supabase PostgreSQL, Neo4j, Qdrant, LangGraph routing, query guardrails, Event Nodes, graph provenance, answer traces, synthetic data, and controlled scenarios. Document processing remains intentionally limited to clean structured contract evidence so the milestone can prove the GraphRAG architecture before adding OCR, noisy documents, invoice processing, the Predictive Engine, or a complete frontend.

**Consequences**: The milestone is large enough to demonstrate the target architecture but bounded enough to avoid mixing graph reasoning, document parsing robustness, forecasting, and frontend polish in the same first delivery.
